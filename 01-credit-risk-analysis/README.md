# Análisis de Riesgo Crediticio y Morosidad

## Contexto
La empresa otorga crédito a clientes sin contar con alertas tempranas ni indicadores claros de riesgo, generando problemas de morosidad y flujo de caja.

## Objetivo
Identificar patrones de atraso en pagos y proponer métricas que permitan anticipar riesgos crediticios.

## Datos
- Base de clientes
- Ventas y cobranzas
- Crédito otorgado  
Datos incompletos y sin estandarización.

## Proceso
- Limpieza de datos con Python
- Análisis exploratorio
- Consultas SQL para métricas clave

## Insights Clave
- Un grupo reducido de clientes concentra gran parte de la deuda vencida
- Existen clientes con atrasos recurrentes

## Recomendaciones
- Implementar un score de riesgo crediticio
- Crear alertas automáticas
- Ajustar políticas de otorgamiento de crédito
